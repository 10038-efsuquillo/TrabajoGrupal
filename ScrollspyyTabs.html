QUE ES XML
XML (Extensible Markup Language) es un lenguaje de marcas diseñado para almacenar y transportar datos de manera estructurada. Se utiliza ampliamente en el intercambio de información entre diferentes sistemas y aplicaciones, especialmente en entornos web.
El XML se basa en etiquetas o marcas que describen los datos contenidos en un documento. Estas etiquetas se utilizan para definir elementos y estructuras de datos, similar a cómo las etiquetas HTML se utilizan para estructurar y presentar contenido en la web. Sin embargo, a diferencia de HTML, XML no tiene etiquetas predefinidas y permite a los usuarios definir sus propias etiquetas personalizadas.

QUE ES JSON
JSON (JavaScript Object Notation) es un formato de intercambio de datos ligero y fácil de leer y escribir. Se basa en la notación de objetos de JavaScript, aunque se ha convertido en un estándar independiente y se utiliza ampliamente en una variedad de aplicaciones y entornos.
JSON se utiliza para representar datos estructurados en forma de pares clave-valor. Los datos se organizan en objetos, que son colecciones no ordenadas de pares clave-valor, y en matrices, que son listas ordenadas de valores. Los valores pueden ser cadenas de texto, números, booleanos, objetos JSON anidados, matrices o el valor nulo.

QUE ES AJAX
AJAX (Asynchronous JavaScript and XML) es una técnica de programación que permite actualizar partes de una página web sin necesidad de recargarla por completo. La característica principal de AJAX es su capacidad para realizar solicitudes asíncronas al servidor, lo que significa que puede enviar y recibir datos en segundo plano sin interrumpir la interacción del usuario con la página.
A diferencia del modelo tradicional de la web, donde cada interacción del usuario implica una carga completa de la página, AJAX permite actualizar y mostrar información adicional en la página sin necesidad de recargarla. Esto mejora la experiencia del usuario al proporcionar una respuesta más rápida y fluida, similar a una aplicación de escritorio.
